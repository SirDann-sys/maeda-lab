# Maeda Lab 🚀

Este repositório contém a infraestrutura e os scripts de automação do meu Home Lab pessoal, carinhosamente apelidado de **Maeda**.

## 📂 Estrutura do Projeto

- **/scripts**: Automações em Bash (Backup, manutenção, etc).
- **/docker**: Configurações de containers (Sunshine, Portainer, etc).
- **/configs**: Arquivos de configuração de sistema e serviços.
- **/devops**: Material de estudo e laboratórios da trilha DevOps (Alura).

## 🛠️ O Hardware (Maeda)
- **CPU:** Intel i7-2600
- **GPU:** NVIDIA GTX 1060
- **OS:** Ubuntu Server
